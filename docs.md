## 🛠️ 快速命令

```bash
# 快速开始
pnpm setup              # 更新占位符信息
pnpm install            # 安装依赖
pnpm build              # 构建项目
pnpm test:all           # 运行所有测试

# 开发
pnpm test:watch         # 监听模式运行测试
pnpm format             # 格式化代码
pnpm format:check       # 检查代码格式

# 发布
pnpm release:patch      # 发布补丁版本
pnpm release:minor      # 发布次版本
pnpm release:major      # 发布主版本
```
